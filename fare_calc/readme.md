# QR Code Generator

QRコードを作成するPythonのプログラムです。

## 環境構築
* Python環境
* Pythonライブラリ
    * BeautifulSoup

## 使い方
1. ライブラリがインストールされていない場合はインストール
    ```sh
    pip install bs4
    ```
2. 以下のコマンドで実行
    ```sh
    python run.py
    ```
    ※経由駅は3つまで指定できます。
