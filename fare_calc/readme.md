# Fare Calc

指定した区間の運賃・定期代を取得し、定期を買うほうが得・損か判定します。

## 環境構築
* Python環境
* Pythonライブラリ
    * BeautifulSoup

## 使い方
1. ライブラリがインストールされていない場合はインストール
    ```sh
    pip install bs4
    ```
2. 以下のコマンドで実行
    ```sh
    python run.py
    ```
    ※経由駅は3つまで指定できます。

## サンクス
* [Yahoo!路線情報](https://transit.yahoo.co.jp/)
